\# Contoso Corp — Windows Server Home Lab



A complete Active Directory home lab built on Windows Server 2022.



\## What Was Built

\- Active Directory with 4 OUs, 12 users, 6 groups

\- 5 Group Policy Objects

\- File Server with NTFS permissions

\- DHCP Server with IP scope 192.168.1.100-200

\- File server NTFS permissions

\- DHCP configuration

\- Windows Server 2022 administration

\- Host: Windows 11 Pro

\- Hypervisor: VirtualBox
